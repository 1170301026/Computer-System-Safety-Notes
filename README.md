# Introduction

## 一.  计算机系统安全问题

​			重要系统中的知名安全缺陷，计算机病毒，计算机蠕虫，分布式拒绝服务攻击，垃圾邮件，僵尸网络，身份窃取，间谍软件等等

## 二. 安全目标

* ##### 机密性Confidentiality(secrecy,privacy)

  授权用户具有读权限

* ##### 完整性Integrity

  仅被授权实体可按所授权限进行修改

* ##### 可用性Availability

  仅被授权实体可访问
  

## 三. 防御方法

* 防止 Prevention
* 阻碍 Hindrance
* 震慑，制止  Deterrence 
* 偏斜，偏差  Deflection
* 检测   Detection
* 恢复   Recovering

## 四.安全原则

* **最弱链接原则** 

* **适当保护原则**

  – 安全目标不是最大化安全，而是最大化实用性，限制风险的花费控制在可接受范围内
  
* **有效性原则**

  – 必须使用控制措施；控制措施要适当、有效；措施要充分、适合、容易使用

  – 用户心理能接受

* **深度防御**

* **晦涩的安全是不起作用的**
